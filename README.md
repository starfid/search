## Starfid Search specifications
- PHP 5.3 - 8, tested.
- Responsive UI.
- PHP Data Objects (PDO).
- Relevance ranked search.
- Multiple tables search.
- Supervised ML dictionaries.
- Dark mode support.
- Multiple view format (HTML & RSS)


## Starfid API
```
?search=word&cat=music&format=rss
```
